БФ “Єдність громади”

Гуманітарна допомога селищам на лінії фронту

Волонтери фонду відправили та відвідали м. Берислав Херсонської області. Люди отримали життєво необхадні ліки, одяг ДЕТАЛЬНІШЕ та речі першої необхідності. Також завезли 80 пакетів з продуктовими наборами для 64 родин. Також схожий груз отримали жителі села Мала Токмачка Запорізької області, села Федорівка Миколаївсьеої області.

БФ “Єдність громади”

З Германії прибули спеціальні ліжка для лікарень

БФ “Єдність громади” отримали та розвезли до лікарень міста 160 ліжок для пацієнтів.ДЕТАЛЬНІШЕ В рамках програми “Допомога лікарням”, 5 лікарень Одеської області отримали новітні ліжка та матраци для своїх пацієнтів. Також з грошової допомоги небайдужих було закуплено спеціальну постільну білизну для цих ліжок.
