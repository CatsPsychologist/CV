ДОПОМОГТИ

Об’єднання фондів «Єдність громади» та «Одеса та одесити» є потужним джерелом допомоги для багатьох людей та для України вже більш ніж 20 років

Фонди засновані у 2001 році. Наша невелика команда - люди, об’єднані спільними цінностями, бажанням перетворювати країну на краще. Працюємо заради спільної перемоги та щасливого майбутнього українського народу.

НОВИНИ ФОНДУ

БФ “Єдність громади”

Гуманітарна допомога селищам на лінії фронту

Волонтери фонду відправили та відвідали м. Берислав Херсонської області. Люди отримали життєво необхадні ліки, одяг ДЕТАЛЬНІШЕ та речі першої необхідності. Також завезли 80 пакетів з продуктовими наборами для 64 родин. Також схожий груз отримали жителі села Мала Токмачка Запорізької області, села Федорівка Миколаївсьеої області.

14.08.2022

БФ “Одеса та Одесити”

Захід для дітей переселенців в кіностудії!

Для маленьких одеситів та гостей міста - дітей переселенців при підтримці Одеської кіностудії ДЕТАЛЬНІШЕ (арт-центру ім. Віри Холодної) волонтери провели розважальний захід. Діти дивилися мультфільми, їли морозиво, танцювали та отримували корисні подарунки

13.08.2022

БФ “Єдність громади”

З Германії прибули спеціальні ліжка для лікарень

БФ “Єдність громади” отримали та розвезли до лікарень міста 160 ліжок для пацієнтів.ДЕТАЛЬНІШЕ В рамках програми “Допомога лікарням”, 5 лікарень Одеської області отримали новітні ліжка та матраци для своїх пацієнтів. Також з грошової допомоги небайдужих було закуплено спеціальну постільну білизну для цих ліжок.
